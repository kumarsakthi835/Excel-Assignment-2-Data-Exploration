
## 📌 Project Title
**Product Dataset Cleaning, Transformation, and Formatting Using Microsoft Excel**

---

## 🎯 Objective
To clean, standardize, transform, and format the product dataset using Microsoft Excel, ensuring data accuracy, consistency, and readiness for analysis.

---

# 🛠️ Tasks Performed

## 1️⃣ Handling Missing Values

### 💰 Price Column
- ✔️ Checked the **Price** column for missing values.
- ✔️ Handled products with missing price information appropriately (e.g., retained for manual validation or updated based on business rules).

### 🗂️ Category Column
- ✔️ Identified missing values in the **Category** column.
- ✔️ Imputed missing categories by referencing similar products or assigning the most appropriate category based on available data.

---

## 2️⃣ Correcting Inconsistent Data

### 📝 Product Name Standardization
- ✔️ Reviewed the **Product Name** column for inconsistent text formats.
- ✔️ Standardized text formatting using the **Find & Replace** feature.

### 🏷️ Category Standardization
- ✔️ Identified typos and spelling errors in the **Category** column.
- ✔️ Corrected and standardized all category names to maintain consistency.

---

## 3️⃣ Removing Duplicates

### 🧹 Duplicate Record Identification
- ✔️ Identified duplicate rows based on the entire dataset.
- ✔️ Removed duplicate records while preserving unique entries.

---

## 4️⃣ Splitting and Merging Data

### ✂️ Split Data
- ✔️ Split the **Product ID** column into:
  - 📅 Manufacturing Date
  - 🌍 Country Code
- ✔️ Removed unnecessary characters from the extracted values, where applicable.

### 🔗 Merge Data
- ✔️ Merged the **Brand Name** and **Product Name** columns.
- ✔️ Created a new column named **Product Brand**.

---

## 5️⃣ Number Formatting

### 💵 Price Formatting
- ✔️ Formatted the **Price** column using the **Currency** number format.

### 📆 Date Formatting
- ✔️ Formatted the **Manufacturing Date** column using the **DD-MM-YYYY** date format.

---

## 6️⃣ Conditional Formatting

### 📊 Price Column
- ✔️ Applied **Data Bars** or **Color Scales** to the **Price** column for better visual representation.

### 🎨 Category Column
- ✔️ Created a custom conditional formatting rule to highlight cells where the **Category** is **"Electronics."**

---

# 🧰 Excel Features Used

- 🔍 Find & Replace
- 🧹 Remove Duplicates
- ✂️ Text to Columns
- 🔗 Merge Text (Concatenate)
- 💲 Currency Formatting
- 📅 Date Formatting
- 🎨 Conditional Formatting
- 📈 Data Cleaning Techniques

---

# ✅ Outcome

- ✔️ Improved data quality by handling missing values.
- ✔️ Standardized text formats and corrected spelling errors.
- ✔️ Removed duplicate records.
- ✔️ Split and merged columns as required.
- ✔️ Applied number and date formatting for consistency.
- ✔️ Enhanced data visualization using conditional formatting.
- ✔️ Prepared the dataset for accurate reporting and further analysis.
